最终决策汇总：
  • 布局：三区一体 — 侧栏（热力图+Commitments）+ 主区（entry list）+ 底部输入
  • 输入：两行 — 上行 item text，下行 token chips + 缺失指示
  • 维度：@ 触发 popover → 选维度（紫色 header + 颜色条）→ 选值（暖灰 header + ← back）→ token 内嵌
  • 时长：item 内自然写 1h 或 # 触发。token 无 # 前缀。删除后回退解析
  • 必选维度：软提示（虚线 chip），不拦截 Enter
  • 编辑：hover 显示 ⋯ → 点击进入 editor（item / duration / chips / Save / Cancel / Delete / +tag）。双击整行也可进入
  • 长文本：item 截断 2 行，goal chip 限宽 100px 省略，sidebar goal 限宽 130px 省略。hover tooltip 看全文
  • 字体：方案 1 — 数字、时间、快捷键用 SF Mono，其余 system-ui
